# MenuAddButton
A `MenuAddButton` is a clickable button to carry out actions like adding and removing items from cart.

# Usage
```js
import { MenuAddButton } from 'react-native-design-system';

<MenuAddButton
  onIncrement={() => handleIncrement())}
  onDecrement={() => handleDecrement())}>
  {this.state.count}
</MenuAddButton>
```
